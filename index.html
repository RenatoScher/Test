Tags Semânticas

O que são?

Tags semânticas são tags que possuem seu propósito ou tipo de conteúdo bem definido, facilitando a visualização e entendimento do código tanto para os programadores quanto para a máquina.

Por que usá-las?

Facilitam a organização do código

Apesar de possuírem o mesmo efeito visual, é muito mais fácil para o programador entender o código quando usamos essas tags, afinal, elas definem bem o seu conteúdo. Aqui um exemplo:

Apenas Divs

Código Semântico

Perceba como fica mais fácil para o “programador” entender e modificar o código quando cada linha tem seu sentido especificado.

Otimizam o funcionamento do site

Essas tags também otimizam o funcionamento do site, o tornando mais acessível para pessoas que usam ferramentas de acessibilidade, como, por exemplo, leitores de tela.

Com o uso de tags que especificam o objetivo de seu conteúdo, essas ferramentas conseguem identificar o que cada parte do texto é. Assim, elas conseguem ler os títulos antes do conteúdo, separar o conteúdo principal do conteúdo secundário, etc.

Melhora o desempenho da página nas ferramentas de pesquisa

Como a máquina consegue entender o conteúdo da página quando ele é mais bem estruturado e hieraquizado, o site passa a subir no “ranking de busca” de ferramentas como o Google ou Firefox, podendo assim aparecer antes de outras que não possuem esse tipo de estrutura.

Quais são essas tags?

Todas as tags semânticas vão ter um aprofundamento maior em seus respectivos tópicos na aba de Tags Estruturais ou na de Tags do Nível do Texto desse artigo

Quando usá-las?

Esse tipo de Tags devem ser usadas sempre que possível, visto que elas só trazem benefícios. Porém, é importante relembrar que elas não devem ser usadas para fins diferentes dos seus, e caso nenhuma tag tenha o fim desejado, não podemos ter medo de usar as tags Div e Span nessas ocasiões.
