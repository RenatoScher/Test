Tipos de Tags Semânticas

Existem 2 tipos de tags semânticas: as Estruturais e as do Nível de Texto. As estruturais servem para estruturar a página visualmente, enquanto as do Nível do Texto para alterar visualmente um texto dentro da página.

Tags do Nivel do texto

<a>

É uma tag usada para definir um Hyperlink, um texto que, ao clique, te redireciona para outra página. O link o qual o texto vai redirecionar o usuário deve ser definido com uma propiedade chamada “href”, caso contrário o conteúdo da tag nem irá aparecer. Caso a tag seja aplicada sem mudanças no CSS, essa tag vai estar sublinhada com uma dessas 3 cores:

  • Azul caso o usuário nunca tenha visitado o link inserido na tag antes;
  • Roxo caso ele já tenha visitado o link inserido na tag antes;
  • Vermelho quando o usuário clica nele, logo antes de ser redirecionado.

<label>

É usado para indicar um texto anexado a outro elemento, (como por exemplo o <input>). Essa tag é diferente de por um h1-h6 do outro elemento pois ao clicar no label, o elemento anexado a ela também é acionado, assim ajudando por exemplo pessoas que tem dificuldade em acertar áreas pequenas do site (como checkboxes).

<em>

Essa tag é usada para emfatizar uma parte do texto do site, mostrando seu conteúdo destacado em itálico. Quando essa tag está presente, leitores de tela vão citar o trecho destacado com ênfase na fala também, deixando assim o texto do site mais acessível para quem precisa usar dessa ferramenta.

<strong>

Ela é usada para definir um trecho do texto que possui uma grande importância para todo o contexto do texto em si. Esta afeta o texto também, destacando seu trecho em negrito.

É importante recitar que esta deve ser usada apenas para trechos importantes, e que caso você queira deixar um texto menos importante em negrito, use CSS ou a tag <b>.

<mark>

Essa tag bota uma parte do texto em realçe, o marcando. Ela faz um efeito parecido como o de um marca-texto escolar, mudando a cor de fundo do texto.

<time>

Ela é usada para definir um horário ou uma data específica. A principal função desse elemento é traduzir seu conteúdo de um jeito que a máquina do usuário possa interpretá-la melhor, traduzindo com perfeição esse trecho para qualquer língua. Ela também possibilita o usuário a adicionar lembretes em seu calendário virtual.

<cite>

Esta define um título de uma obra de arte ou de literatura, como livros, poemas, músicas, esculturas, etc. Esse trecho é renderizado em itálico, e a tag não deve ser utilizada para citar o nome do autor.

<q> & <blockquote>

As tags <q> e <blockquote> são usadas para definir uma citação diretas, onde <q> é para citações curtas e <blockquote> para citações longas. *Imagem* Note que a tag <q> são usadas dentro do trecho do texto, e as aspas ao seu redor são geradas automaticamente. Agora a tag <blockquote> cria um novo parágrafo, com recuo e espaçamento automático.

<small>

Essa tag é usada para formatar textos com um tamanho de fonte pequeno, como abas de copyright, comentários entre o texto ou trechos menos importantes. Sua formatação apenas transforma o texto com a fonte “smaller”, então é possível alcançar o mesmo resultado com modificações no CSS.
